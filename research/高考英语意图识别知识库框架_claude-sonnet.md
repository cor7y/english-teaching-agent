# 高考英语学情意图识别：多维度知识库框架

> 面向高中英语（高考）学生端，用于 Agent 意图识别与学情归纳分类
> 
> 版本 v1.0 | 调研日期：2026年4月

---

## 框架总览

学生的输入文本可能横跨以下 **6 个一级维度**，每个维度下有 2-5 个二级分类，每个分类包含具体问题标签（意图节点）。

```
Dimension 1  语言知识       → 词汇 / 语法
Dimension 2  四大核心技能   → 听力 / 阅读 / 写作 / 口语
Dimension 3  高考专项题型   → 完形 / 七选五 / 语法填空 / 短文改错
Dimension 4  应试策略       → 时间管理 / 审题习惯 / 错误分析
Dimension 5  学习行为       → 学习方法 / 练习习惯 / 错题管理
Dimension 6  心理与动机     → 焦虑 / 自信 / 动机 / 元认知
```

意图识别的目标：将学生的自由文本输入映射到上述节点，单条输入可以命中 **多个节点（多标签分类）**。

## Dimension 1：语言知识（Language Knowledge）

### 1.1 词汇（Vocabulary）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 词汇量不足 | "单词太少"、"读文章碰到很多不认识的词" | 主动词汇量低于 3500 高频词范围，影响阅读和写作 |
| 单词记不住 | "背了就忘"、"记了好多遍还是不会" | 记忆策略单一，靠机械重复，缺乏语境激活 |
| 熟词僻义 | "单词认识但句子看不懂"、"这个词我知道但这里啥意思" | 常见词在特定语境下有偏僻含义（如 subject/company/company 的动词用法） |
| 词形变化 | "不知道这里填名词还是形容词"、"派生词不会" | 词性转换（动→名→形→副），词缀规律掌握不足 |
| 词汇搭配 | "搭配总记不住"、"什么词后面接什么介词" | 固定搭配（collocations）和短语动词（phrasal verbs）记忆混乱 |
| 写作词汇贫乏 | "作文老用那几个词"、"nice/good/said 换不掉" | 输出词汇单调，缺乏高级替换词和情感动词储备 |

**重点补充：高频失分词汇区**
- 熟词僻义高频词：subject（受试者）、present（出席）、address（演讲/处理）、company（陪伴，v.）、observe（遵守）
- 情感动词储备：表"说"（whisper/snap/murmur）、表"哭"（sob/weep）、表"笑"（grin/chuckle）
- 词根前缀掌握：un-/dis-/mis-、-ful/-less/-ness/-tion/-ity

### 1.2 语法（Grammar）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 时态/语态 | "时态总搞混"、"主动被动分不清" | 过去完成时、现在完成时混用；被动语态构成不熟 |
| 非谓语动词 | "不定式分词老搞错"、"to do/doing/done 不会选" | 四种非谓语（to do/doing/done/do）的选择规则掌握不牢，是语法填空最高频失分点 |
| 定语从句 | "关系代词不会选"、"which/that/who 搞混" | 限制性/非限制性从句区分；关系代词与关系副词区分（where/when/which） |
| 名词性从句 | "whether/if 不知道怎么用" | 主语从句、宾语从句、表语从句、同位语从句混淆 |
| 状语从句 | "连词不会选"、"让步从句/条件从句不清楚" | 连词语义（though/unless/as long as/once）区分不清 |
| 虚拟语气 | "虚拟语气总错"、"wish/if 后面的时态乱" | 与现在/过去/将来事实相反的三种虚拟语气结构 |
| 特殊句式 | "强调句/倒装句不会" | It is...that 强调句；否定词前置引起的部分倒装（Never have I...） |
| 冠词 | "a/an/the 不知道填哪个" | 第一次提及/再次提及规则；习惯用法中的零冠词 |
| 主谓一致 | "单数复数不对" | 就近/就远原则；集合名词处理 |

**高频语法考点优先级（按失分量排序）：**
1. 非谓语动词（语法填空必考）
2. 时态与语态
3. 定语从句引导词
4. 连词与从句关系
5. 冠词

## Dimension 2：四大核心技能（Core Language Skills）

### 2.1 听力（Listening）

高考听力满分 30 分（全国卷），分 5 段共 25 题。

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 语速跟不上 | "听的时候太快了跟不上"、"反应不过来" | 短时记忆处理速度不足，无法边听边理解 |
| 细节捕捉失误 | "明明听到了但还是选错"、"数字地点老出错" | 对时间、数字、地点等具体信息的同义替换辨别不足 |
| 主旨把握困难 | "不知道在说什么"、"整体意思没听懂" | 缺乏听力宏观预测意识，不会利用首句定位主题 |
| 推断判断失误 | "推断题老错"、"态度题不会" | 无法从语气、情态词、言外之意推断说话者立场/意图 |
| 听力心态问题 | "一紧张就听不进去"、"没听到一点就放弃了" | 抑制型焦虑导致听力过程中注意力崩溃 |
| 听力练习方法问题 | "做完就过了不知道怎么复习" | 缺乏精听复盘习惯，做题后不对答案原文对照 |

**听力核心题型分布：**
- 主旨大意题（约2题）
- 细节理解题（约8题，最多）
- 推断判断题（约5题）
- 人物关系/场合推断（约5题）
- 数字运算/信息比较（约5题）

---

### 2.2 阅读（Reading）

阅读理解占 40 分（4篇文章 × 10 分），是最大分值板块。

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 阅读速度慢 | "时间不够用"、"还没读完就没时间了" | 阅读策略单一（逐词翻译），无法跳读/略读定位 |
| 生词干扰 | "碰到不认识的词就卡住了" | 不会利用上下文猜测词义，遇生词停顿影响整体理解 |
| 细节题定位失误 | "去文章找了但找错地方了" | 关键词定位意识弱，答案段落识别不准 |
| 推断题失分 | "推断题老选错"、"总选过度推断的选项" | 无法区分文本明示信息与个人主观添加信息 |
| 主旨/标题题 | "不知道文章在说什么"、"主旨题老选错" | 缺乏快速提炼段落主题句和全文核心议题的能力 |
| 词义题 | "划线词语意思猜不出来" | 词义猜测技巧（词根、上下文语义场）运用不足 |
| 态度/观点题 | "作者态度题不会" | 无法识别作者立场词（skeptical/optimistic/concerned） |
| 长难句理解 | "句子太长看不懂" | 缺乏句子主干识别能力，无法剥离多层修饰成分 |

**阅读文章类型（高考分布）：**
- A篇：实用类（招募广告、活动通知、旅游信息），难度最低
- B篇：记叙文/人物故事，情节理解为主
- C篇：说明文/科普，逻辑推断为主
- D篇：议论文/学术语篇，推断+作者意图，难度最高

### 2.3 写作（Writing）

新高考写作总分 40 分，分两部分：
- **应用文（小作文）**：15 分，80词左右
- **读后续写（大作文）**：25 分，150词左右

#### 2.3.1 应用文（Functional Writing）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 文体格式不规范 | "不知道书信格式"、"开头结尾怎么写" | 书信称谓、落款、语气（正式/半正式）掌握不足 |
| 要点遗漏 | "总忘写某一个要点" | 审题不仔细，对题目要求的覆盖不完整 |
| 语言单调 | "句型太简单"、"老师说句式太单一" | 写作词汇和句式缺乏多样性，依赖简单句 |
| 连接不自然 | "感觉文章不连贯" | 缺乏衔接词（therefore/in addition/as a result）的灵活运用 |
| 常见应用文类型 | — | 书信（建议信/感谢信/投诉信）、通知、发言稿、推荐信、求助信 |

#### 2.3.2 读后续写（Continuation Writing）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 情节脱离原文 | "感觉和原文没关系"、"老师说融洽度低" | 续写逻辑与原文情节、人物、风格脱节（融洽度是核心评分项） |
| 不会用关键词 | "不知道怎么用划线词"、"强行用词很奇怪" | 未能将原文标记关键词自然融入续写（要求使用5个以上） |
| 细节描写薄弱 | "故事太干"、"老师说没有画面感" | 缺乏人物动作/心理/环境的细节刻画（"show, not tell"意识缺失） |
| 段落失衡 | "两段字数差很多" | 两个续写段落严重不对等，情节处理轻重失调 |
| 主题跑偏 | "故事走向歪了"、"老师说价值观有问题" | 对原文主题/价值观误判，续写走向悲观/荒诞/无意义 |
| 时态人称混乱 | "时态一直错"、"he/she 用混了" | 记叙文时态（过去时为主）和人称未能保持一致 |
| 对话滥用 | "对话太多但没实质内容" | 过度使用对话填充篇幅，缺乏叙述性语言和高级词汇展示 |

**读后续写评分四维度：**
1. 与原文的融洽度（情节/人物/风格衔接）
2. 内容丰富性+关键词覆盖率（≥5个）
3. 词汇语法的准确性与多样性
4. 语篇连贯性（段落间逻辑+连接成分）

## Dimension 3：高考专项题型（Exam-Specific Question Types）

### 3.1 完形填空（Cloze Test）

高考完形满分 15 分，15 空，以记叙文/夹叙夹议为主，考查词汇在语境中的运用。

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 整体理解困难 | "完形不知道在说什么" | 未能先快速浏览获取主旨，逐空作答导致误判 |
| 情感倾向失误 | "感觉意思对但就是选错" | 未识别文章的情感走向（转折/递进），导致关键词选择与语气矛盾 |
| 近义词辨析 | "四个选项意思都差不多不知道选哪个" | 无法根据动宾/主谓搭配、语义精度区分近义词 |
| 连词辨析 | "连词那种题老错" | 表逻辑关系的连词（对比/递进/让步/结果）语义不熟 |
| 做题策略缺失 | "不会做完形" | 不知道三遍读文法：首遍速读→二遍细读→三遍验证 |

---

### 3.2 七选五（Five-from-Seven）

满分 12.5 分，5 空，考查语篇逻辑和段落衔接。

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 段落逻辑不清 | "七选五完全不会" | 无法分析段落内部论证结构（总分/因果/对比） |
| 代词指代问题 | "代词指的是什么不知道" | 未利用选项/文中代词（it/they/this/these）反向定位 |
| 主题句识别 | "不知道应该填主题句还是例子" | 无法区分空格前后需要的是论点句、例子句还是总结句 |
| 关键词复现 | "不会用关键词定位" | 忽略空格前后与选项之间词语/主题的重复/同义呼应 |

---

### 3.3 语法填空（Grammar Filling）

满分 15 分，10 空，分纯空格（3-4题，考虚词）和提示词题（6-7题，考实词变形）。

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 纯空格不知填什么词性 | "这个空不知道填什么" | 未判断句子成分缺失类型（缺连词/代词/冠词/介词） |
| 提示词词形判断错误 | "括号给了词但不知道怎么变" | 词性转换判断失误（名→形/动→名/形→副）；时态语态分析错 |
| 非谓语形式错误 | "to do 还是 doing 不确定" | 非谓语动词选择：与主句主语的主被动关系判断失误 |
| 从句引导词 | "引导词用 which 还是 where/when" | 关系词引导定语从句/状语从句时的语法功能区分 |
| 拼写错误 | "意思对但词拼错了" | 词形变化（单复数、过去式不规则变形）拼写不稳定 |

**语法填空纯空格高频考点：**
冠词（a/an/the）> 介词 > 连词（并列/从属）> 代词 > 从句引导词（which/that/when/where）

### 3.4 短文改错（Error Correction）
*注：新高考（部分省份）已取消此题型，替换为读后续写。保留为潜在节点。*

| 意图节点 | 核心问题描述 |
|---|---|
| 动词形式错误 | 时态/语态/非谓语形式 |
| 名词数/冠词 | 单复数/可数不可数/冠词增删 |
| 代词错误 | 人称/格/指代一致性 |
| 连词/逻辑词错误 | and→but；although→so 等逻辑关系错误 |
| 多词/漏词 | 句子成分多余或缺失 |

## Dimension 4：应试策略（Test-Taking Strategies）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 时间管理 | "考试时间不够"、"作文没时间写完" | 各题型时间分配策略缺失（听力→阅读→七选五→完形→语法填空→作文） |
| 审题失误 | "题目看错了"、"答的不是题目问的" | 未细读题干，对"作者观点"vs"文章细节"等提问方式不敏感 |
| 答题顺序 | "不知道先做哪道题" | 无个性化顺序策略，导致强项题型时间不足 |
| 蒙题/猜题 | "不会的题怎么猜" | 对完全不确定的题目无合理排除逻辑 |
| 书写规范 | "老师说卷面扣分"、"字写太乱" | 书写不规范影响主观题评分印象分 |

**建议时间分配参考（全国卷 120 分钟）：**

| 题型 | 建议时间 |
|---|---|
| 听力 | 约 30 分钟（随录音固定） |
| 阅读理解（4篇） | 约 35 分钟 |
| 七选五 | 约 10 分钟 |
| 完形填空 | 约 15 分钟 |
| 语法填空 | 约 10 分钟 |
| 应用文写作 | 约 10-12 分钟 |
| 读后续写 | 约 28-30 分钟 |

## Dimension 5：学习行为（Learning Behaviors）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 学习方法低效 | "我已经很努力了但分数不涨"、"刷了很多题没用" | 学习行为量大但质低，缺乏针对性复盘和策略调整 |
| 错题管理缺失 | "不做错题本"、"错了再做还是错" | 做错题后不分析根本原因，无法建立个人错误模式认知 |
| 单词记忆方法单一 | "就是背单词书"、"死记硬背" | 依赖孤立记忆，缺乏语境输入（阅读/听力）的词汇复现强化 |
| 阅读量不足 | "课外很少读英语" | 语感培养依赖课堂，课外英语输入极少 |
| 写作练习回避 | "不喜欢写作文"、"觉得提升慢所以不练" | 因产出困难而回避写作练习，形成恶性循环 |
| 听力练习不系统 | "听力就是考试前刷题" | 缺乏日常精听习惯，听力基础积累薄弱 |
| 复习无计划 | "不知道先复习什么"、"乱刷题" | 缺乏按题型/知识点分优先级的系统复习规划 |
| 过度依赖答案解析 | "做完直接看答案"、"不先思考" | 刷题时直接翻答案，不形成自主解题思路 |

## Dimension 6：心理与动机（Psychology & Motivation）

### 6.1 学习焦虑（Anxiety）

研究显示，超过 70% 的高中生在英语学习中存在不同程度的心理障碍（焦虑/自卑/恐惧/厌学）。焦虑可分为：
- **抑制型焦虑（Debilitating）**：产生负面影响，需干预
- **促进型焦虑（Facilitating）**：适度紧张，可转化为动力

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 考试焦虑 | "一考试就脑子空白"、"平时会做考试错" | 测试焦虑导致已掌握知识的提取失败 |
| 听力焦虑 | "听的时候心跳加速"、"听到一句没听到就放弃了" | 输入焦虑：对语速和生词的恐惧引发注意力崩溃 |
| 写作焦虑 | "不知道从哪里写起"、"一写就头脑空白" | 输出焦虑：对表达不准确的恐惧导致写作启动困难 |
| 慢性挫败感 | "英语学了很久一直没提高"、"觉得自己不是学英语的料" | 长期低分导致自我效能感（self-efficacy）损耗 |

### 6.2 动机（Motivation）

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 内在动机低 | "不喜欢英语"、"觉得英语没意思" | 工具性动机（高考）主导，缺乏对语言本身的兴趣 |
| 目标模糊 | "不知道该达到什么水平"、"英语多少分算好" | 没有明确的分数目标或题型目标，学习缺乏锚点 |
| 成就感缺失 | "总感觉学了没进步"、"看不到提升" | 进步不可见导致持续性动力下降 |

### 6.3 元认知（Metacognition）

元认知即学生对自身学习状态的认知和调节能力，是学情备忘录数据捕捉的关键层。

| 意图节点 | 典型输入信号 | 核心问题描述 |
|---|---|---|
| 问题归因不准 | "不知道为什么老错"、"我明明会的为什么会错" | 无法识别失分的真实原因（知识漏洞 vs 策略问题 vs 心态问题） |
| 薄弱点认知模糊 | "感觉哪里都不好"、"不知道该先补什么" | 没有清晰的个人知识地图，无法优先级排序学习任务 |
| 过度自信偏差 | "这个我会了"（实际没掌握）| 主观感受的掌握度与客观测试表现不符 |
| 学习策略反思缺失 | "一直这么学的，不知道对不对" | 从未反思现有学习方法是否有效，机械重复低效行为 |

## 附录 A：维度间关联关系

```
词汇不足 ──→ 阅读速度慢、写作词汇贫乏、完形失误率高
语法薄弱 ──→ 语法填空失分、写作语法错误、长难句理解困难
阅读能力弱 ──→ 读后续写融洽度低（读懂是续写前提）
元认知模糊 ──→ 学习行为低效（不知道补什么）
学习焦虑 ──→ 听力焦虑、考试焦虑、写作启动困难
```

## 附录 D：设计建议（意图识别 Prompt 方向）

1. **多标签输出**：单条输入应允许同时输出 2-4 个意图节点，学生问题往往是交叉的（如"词汇弱→阅读差→写作差"是一个连锁）

2. **区分深度**：
   - **表层意图**（学生明确说的）：直接映射到节点
   - **隐含意图**（从描述推断的）：例如"刷了很多题没用"背后是`学习方法低效`+`元认知→问题归因不准`
   - **情绪信号**：如有焦虑/沮丧词语，同步标记心理维度节点

3. **置信度分级**：对明确命中的节点给高置信度；对推断性节点给中置信度，可在学情备忘录中标注为"待核实"

4. **节点优先级排序**：高考直接失分节点（阅读/写作/语法填空）>间接影响节点（学习习惯/心理）；MVP 阶段优先覆盖前 3 个 Dimension
