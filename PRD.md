# ETA 产品需求文档

## 产品定位

面向高中生的英语学习Agent，以**学情备忘录**为核心，通过意图识别将学生的自由提问结构化，持续积累学情数据，形成"提问→识别→记录→教学→自测→反馈"的闭环。

竞品参考：精准学"寒雪老师"（存在严重体验缺陷，见NotebookLM调研）。
ETA不是对标竞品做全量替代，而是从一个可验证的最小闭环切入。

## MVP范围

### 聚焦题型：语法填空

选择理由：
- 闭环最短：答案唯一，对错清晰，mastery可量化
- 覆盖语法分类体系的大部分子类，能验证分类框架的可用性
- 高考占15分，学生普遍得分率60-70%，有明确提分空间

### 功能架构

```
用户文本输入
  → 意图识别（6类分流：K/D/S/M/P/E）
    → 知识维度定位（语法2.1-2.10 + 词汇1.3）
      → 能力层级判断（L1-L4）
        → 录入学情备忘录
          → [教学响应] ← MVP暂不做深度教学
            → 用户自测
              → 自动反馈 → 更新学情备忘录
                → 闭环 ✓
```

### 意图分类（第一层）

| 代码 | 类型 | 典型输入 | 学情记录 |
|------|------|---------|---------|
| K | 知识查询 | "定语从句怎么用" | 标记知识点为未掌握/模糊 |
| D | 错误诊断 | "这个句子哪里错了" | 记录错误类型和涉及知识点 |
| S | 解题求助 | "这道完形填空怎么做" | 记录题型+失分点+策略缺陷 |
| M | 方法策略 | "怎么背单词更有效" | 记录学习策略短板 |
| P | 练习请求 | "给我出几道语法填空" | 记录练习偏好和自测结果 |
| E | 情感表达 | "英语太难了学不会" | 标记情感状态，触发鼓励 |

### MVP知识维度范围

- 语法：2.1时态语态、2.2非谓语、2.3从句、2.4特殊句式、2.5主谓一致、2.6冠词介词、2.7代词、2.8情态动词、2.9比较级最高级、2.10名词形态
- 词汇：1.3词性与构词法
- 教材范围：人教版（2019）必修1-3 + 选择性必修1-3

## 学情备忘录

核心数据结构见 `knowledge-base/05-意图识别与学情备忘录设计.md`。

关键设计：
- 每个知识点维护 mastery 值（0.0-1.0）和 status 状态机（unknown→learning→mastered/weak）
- 答对+0.1，答错-0.15，连续3对→mastered，连续2错→weak
- 7天未练习 mastery 自然衰减 -0.05/周
- 错误历史记录具体错误类型、语境、涉及知识点

## 验证指标

| 指标 | 目标 |
|------|------|
| 意图识别准确率（6类） | >85% |
| 知识维度定位准确率 | >80% |
| 学情备忘录更新一致性 | 每次交互后状态正确更新 |
| 闭环完整性 | 输入→识别→记录→反馈→推荐全链路跑通 |

## 开发阶段

### Phase 1：意图识别PoC（当前）
验证：用户文本输入 → 正确分流为6类意图 + 定位知识维度 + 判断能力层级

### Phase 2：学情备忘录 + 闭环
将识别结果写入学情备忘录，实现mastery追踪和状态更新

### Phase 3：自测 + 反馈
生成针对性练习题，自测结果自动回写学情备忘录

### Phase 4：教学响应（待定义）
基于学情的个性化教学内容生成
