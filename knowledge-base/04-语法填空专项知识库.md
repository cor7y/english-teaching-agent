# 语法填空专项知识库（MVP核心）

本文档是教学Agent意图识别和学情诊断的核心参考，覆盖语法填空题型涉及的全部知识点、常见错误模式和诊断逻辑。

信息来源：《普通高中英语课程标准（2017年版2020年修订）》；人教版（2019）高中英语教材语法点分布；高考英语全国卷/新高考卷真题分析（2020-2024）；一线教师教学经验总结

---

## 一、语法填空题型特征

### 基本信息
- 题量：10题/篇
- 分值：每题1.5分，共15分
- 形式：一篇200-250词的短文，设10个空
- 两种题型：
  - 有提示词（约6-7题）：给出动词/形容词/名词原形，需变形
  - 无提示词（约3-4题）：需填连词/介词/冠词/代词

### 考点分布规律（每年稳定）
- 动词时态/语态：1-2题
- 非谓语动词：1-2题
- 从句连接词：1-2题
- 词性转换：2-3题
- 冠词/介词/代词：1-2题
- 比较级/最高级：0-1题

## 二、各考点详细知识与常见错误

### 考点1：动词时态与语态（→ 分类体系 2.1）

#### 知识要点
8种核心时态：
- 一般现在时：do/does — 表习惯、事实、真理
- 一般过去时：did — 表过去发生的动作或状态
- 一般将来时：will do / be going to do — 表将来的计划或预测
- 现在进行时：am/is/are doing — 表正在进行的动作
- 过去进行时：was/were doing — 表过去某时正在进行的动作
- 现在完成时：have/has done — 表过去发生但与现在有联系的动作
- 过去完成时：had done — 表"过去的过去"
- 现在完成进行时：have/has been doing — 表从过去持续到现在的动作

被动语态构成：be + 过去分词
- 一般现在时被动：am/is/are + done
- 一般过去时被动：was/were + done
- 现在完成时被动：have/has been + done
- 含情态动词被动：can/should/must + be + done

#### 判断方法
1. 先判断该空是否需要谓语动词（句子是否缺谓语）
2. 看时间状语确定时态（yesterday→过去时，since→完成时，now→进行时）
3. 看主语与动词的关系确定语态（主语是动作执行者→主动，主语是动作承受者→被动）
4. 注意上下文时态一致性

#### 常见错误
- 一般过去时与现在完成时混淆（最高频）
  - 错误模式：有since/for/already等标志词时仍用一般过去时
  - 错误模式：有yesterday/last year等明确过去时间时用现在完成时
- 主谓不一致
  - 错误模式：主语是第三人称单数时忘记加-s/-es
  - 错误模式：主语是不可数名词或each/every开头时用复数动词
- 被动语态遗漏
  - 错误模式：主语是动作承受者时仍用主动语态

### 考点2：非谓语动词（→ 分类体系 2.2）

#### 知识要点
三种形式：
- 不定式（to do）：表目的、将来、具体的一次性动作
- 动名词（doing）：表习惯、一般性动作、已发生的事
- 过去分词（done）：表被动、完成

核心用法：
- 作主语：Doing/To do sth is important.
- 作宾语：enjoy doing / want to do / have sth done
- 作定语：the ability to do / the boy standing there / the broken window
- 作状语：To improve English, he reads every day. / Seeing the teacher, he stood up. / Moved by the story, she cried.
- 作补语：see sb doing/do/done

#### 判断方法
1. 先确认该空不是谓语（句子已有谓语动词）
2. 看逻辑主语与动词的关系：
   - 主动关系 → doing（进行/主动）或 to do（目的/将来）
   - 被动关系 → done
3. 看语境：
   - 表目的 → to do
   - 表原因/伴随/时间 → doing（主动）/ done（被动）
   - 在介词后 → doing

#### 常见错误（错误率最高的考点，约40-50%）
- to do/doing/done三选一判断失误
  - 错误模式：不看逻辑主语，凭感觉选择
  - 错误模式：介词后用to do而非doing
  - 错误模式：被动关系时用doing而非done
- 固定搭配记忆不牢
  - 错误模式：enjoy to do（应为enjoy doing）
  - 错误模式：suggest to do（应为suggest doing）

### 考点3：从句连接词（→ 分类体系 2.3）

#### 知识要点

定语从句连接词选择：
- 先行词是人：who/whom/whose/that
- 先行词是物：which/that/whose
- 先行词是时间：when（= at/in/on which）
- 先行词是地点：where（= at/in/on which）
- 先行词是原因：why（= for which）
- 非限制性定语从句不用that

名词性从句连接词选择：
- 主语从句：what/that/whether/who/how/why等
- 宾语从句：that（可省略）/what/whether/if/who/how等
- 表语从句：that/what/whether/because/as if等
- 同位语从句：that（最常见）/whether/what/how等

状语从句连接词选择：
- 时间：when/while/as/before/after/since/until/as soon as
- 原因：because/since/as/now that
- 条件：if/unless/as long as/provided that
- 让步：although/though/even though/while
- 目的：so that/in order that
- 结果：so...that/such...that
- 比较：as...as/than
- 方式：as/as if

#### 判断方法
1. 先判断从句类型（定语从句看有无先行词，名词性从句看从句在句中的成分）
2. 定语从句：看先行词 + 从句中缺什么成分
3. 名词性从句：看从句是否完整（完整用that/whether，缺成分用what/who/how等）
4. 状语从句：看逻辑关系

#### 常见错误
- 定语从句中which/that/who选择错误
  - 错误模式：非限制性定语从句中用that
  - 错误模式：先行词是人时用which
- 名词性从句中that/what混淆
  - 错误模式：从句缺主语或宾语时用that（应用what）
  - 错误模式：从句完整时用what（应用that）
- 状语从句连接词逻辑关系判断错误

### 考点4：词性转换（→ 分类体系 1.3）

#### 知识要点

常见转换规则：
- 动词→名词：-tion/-sion（educate→education）、-ment（develop→development）、-ance/-ence（appear→appearance）、-er/-or（teach→teacher）
- 形容词→名词：-ness（happy→happiness）、-ty/-ity（safe→safety）、-th（grow→growth）
- 名词→形容词：-ful（care→careful）、-less（hope→hopeless）、-ous（danger→dangerous）、-al（nature→natural）、-ive（create→creative）
- 形容词→副词：-ly（quick→quickly）
- 动词→形容词：-ing（interest→interesting）、-ed（interest→interested）、-able/-ible（accept→acceptable）

#### 判断方法
1. 看空格在句中的位置和功能
2. 在名词前/作定语 → 需要形容词
3. 修饰动词/形容词/句子 → 需要副词
4. 作主语/宾语 → 需要名词
5. 注意不规则变化

#### 常见错误
- 不规则变化记忆不牢
  - 错误模式：possible→possiblely（应为possibly）
  - 错误模式：true→truely（应为truly）
- -ing/-ed形容词混淆
  - 错误模式：I am interesting in English.（应为interested）
- 忘记双写/去e等拼写规则

### 考点5：冠词（→ 分类体系 2.6）

#### 知识要点
- a/an：泛指，首次提到，表"一个"
- the：特指，上文提到过，独一无二，最高级前，序数词前
- 零冠词：不可数名词/复数名词泛指，专有名词，三餐/球类/学科前

#### 判断方法
1. 看名词是否可数、单复数
2. 看是泛指还是特指
3. 注意固定搭配（in the morning, by bus, at school等）

#### 常见错误
- 该用the时用a/an或不用
- 不可数名词前误加a/an
- 元音音素前用a而非an（如an hour, a university）

---

### 考点6：介词（→ 分类体系 2.6）

#### 知识要点
- 时间介词：at（时刻）、on（具体日期）、in（年/月/季节/世纪）、for（时间段）、since（时间点）、during（期间）
- 地点介词：at（小地点）、in（大地点/内部）、on（表面）、to（方向）
- 其他常用：by（方式/截止时间）、with（伴随/工具）、of（所属）、about（关于）

#### 常见错误
- 中式搭配直译（arrive school → arrive at school）
- 固定搭配记忆不牢（be interested in, be good at, depend on等）

---

### 考点7：代词（→ 分类体系 2.7）

#### 知识要点
- 人称代词：主格/宾格选择
- 物主代词：形容词性/名词性选择
- 反身代词：主语和宾语是同一人时使用
- 指示代词：this/that/these/those
- it的特殊用法：形式主语、形式宾语、强调句

#### 常见错误
- 指代不一致（前面用they，后面用it）
- 形容词性物主代词和名词性物主代词混淆

---

### 考点8：比较级与最高级（→ 分类体系 2.9）

#### 知识要点
- 规则变化：-er/-est，more/most
- 不规则变化：good→better→best，bad→worse→worst，many/much→more→most
- 比较级标志词：than, much/far/even + 比较级
- 最高级标志词：the + 最高级 + of/in

#### 常见错误
- 不规则变化记忆不牢
- 比较级前误加the（除特定结构外）
- 最高级前忘加the

## 四、学情备忘录更新逻辑（语法填空场景）

### 触发条件
1. 学生做完一道/一组语法填空题后
2. 学生提问涉及某个语法知识点后
3. 学生请求诊断某个错误后

### 更新规则
- 答对：该知识点mastery值上调（+0.1，上限1.0）
- 答错：该知识点mastery值下调（-0.15），记录具体错误类型到error_history
- 连续3次答对同一知识点：标记为"已掌握"
- 连续2次答错同一知识点：标记为"重点薄弱"，触发专项讲解
- 超过7天未练习某知识点：mastery值自然衰减（-0.05/周）

### 反馈生成
- 基于mastery值排序，优先推荐最薄弱的知识点进行练习
- 每次练习后生成简短反馈："本次语法填空10题中，你在非谓语动词（2题错误）和时态判断（1题错误）上需要加强"
- 周期性生成学情报告："本周练习15组语法填空，整体正确率从60%提升到72%。非谓语动词仍是最大薄弱点（正确率45%），建议重点复习to do/doing/done的选择规则"
